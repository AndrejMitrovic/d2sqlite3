    Interface IUnlockNotifyHandler

    UnlockNotifyHandler interface to be used for custom implementations of UnlockNotify pattern with SQLite.

    interface IUnlockNotifyHandler ;

    Properties

    NameTypeDescription
    result[get] intResult after wait is finished

    Methods

    NameDescription
    emit Unlocks the handler. This is called from registered callback from SQLite.
    emit Unlocks the handler. This is called from registered callback from SQLite.
    reset Resets the handler for the next use
    wait Blocks until emit is called
    wait Blocks until emit is called

    Note

    For the C API sqlite3_unlock_notify to be used, this library must be compiled with -version=SqliteEnableUnlockNotify. Otherwise only emulated solution is provided, that is based on retries for the defined amount of time.

    Implementation must be able to handle situation when emit is called sooner than the wait itself.
